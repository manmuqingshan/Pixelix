Icons

Overview

Upload

Supported are bitmap (.bmp) and GIF (.gif) files.

The bitmap files are limited to:

  • 24 or 32 bits per pixel.
  • 1 plane.
  • No compression.

Note, if you are using _gimp_ to create bitmap files, please configure like:

  • Compatibility options: Don't write color informations.
  • Extended options: Select 24 bit per pixel.

LaMetric Icons

Have a look to the LaMetric icon overview and enter the icon id below.

Only icons in the GIF format are supported.

After you entered the icon id, first click on the preview button. If the preview is successful, you can then click the download button to fetch the icon.
