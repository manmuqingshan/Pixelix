DateTimePlugin

This plugin lets you see the current date and time for any time zone you choose.

With the display mode, it can be configured to show only the date or only the time as well.

It can also be set to show the time in 12-hour or 24-hour format.

Additionally, the time zone can be specified to display the time in a particular region. If no time zone is set, the configured time zone in the settings will be used.

Depending on the country, the first day of the week can be configured.

With the time and date format specifier, the output format can be customized.

The format is according to strftime(). For colorization text properties can be added.

  • %b - Abbreviated month name (3 characters)
  • %d - Day of the month, zero-padded (01-31)
  • %m - Month as a decimal number (01-12)
  • %H - Hour in 24h format (00-23)
  • %I - Hour in 12h format (01-12)
  • %M - Minute (00-59)
  • %p - AM or PM designation
  • %Y - Year

Configuration

Display

The time format string defines how the time is shown.
The date format string defines how the date is shown.
If empty, the time zone from settings will be applied.
